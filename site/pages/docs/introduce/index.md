### 介绍

#### 引入

```sh 
  # 安装
  npm install lixi-ui-react
  npm install lixi-cli -g // 全局安装
  lixi-cli my-app // 就可以创建自己的项目了
  cd my-app
  npm install
  npm start
```

### 目录结构说明

```js
  |- material ------------------- 物料库
  |- public --------------------- 公共文件
  |- script --------------------- 启动脚本
  |- server --------------------- 后台服务项
  |- site ----------------------- 网站的配置项
  |- test ----------------------- 单元测试
  |- .babelrc ------------------- babel配置项
```
